              // the first 6 bits are 0, so the result will have 6 bits - 000000, the 7th and 8th bit will be 1 because
              // both the 2 and 1 have a bit at the end. - the result is 00000011 //console log = 3

              // 1 = 00000001  - 8 bits, 1 byte
              // 2 = 00000010
